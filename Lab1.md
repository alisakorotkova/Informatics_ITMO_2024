*С чего начинается Linux? С Терминала!!!*

Итак, вы установили дистрибутив Linux Ubuntu (жедательно его, но если у вас уже есть другой Linux или вообще MacOS то можно их использовать) и запустили его. Перед вами рабочий стол, панель задач, иконки и прочие вещи - все как и в других ~~менее удобных~~ операционных системах. Предвижу, что вы уже вдоволь наигрались с новым (или нет?) для вас графическим интерфейсом (GUI - graphical user interface) и готовы к самому интересному - знакомству с Терминалом (CLI - command line interface) и выполнению Лабораторной работы №1.

1. Создайте новый файл с именем `script.bash`

```bash
touch script.bash
```

2. Откройте созданный файл `script.bash` для редактирования. Стандартный текстовый редактор в Linux Ubuntu это `gedit`. Выполните в терминале

```bash
gedit script.bash
```

3. Впишите следующий скрипт

```bash
#!/bin/bash

echo "Welcome to ITMO University"
```
